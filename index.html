📖 ความหมายสถานะ:
🟢 Active = เปิดล่าสุด < 30 นาที — ส่ง notification ได้แน่นอน
🟡 Recent = 30 นาที - 2 ชม — น่าจะส่งได้
🟠 Stale = 2-6 ชม — iOS อาจจะ hibernate แล้ว
🔴 Hibernate = 6-12 ชม — ส่งไม่ถึงแน่นอน ต้องให้ user เปิดแอปใหม่
⚫ Dead = > 12 ชม — token หมดอายุ ระบบจะลบเมื่อ user เปิดใหม่
